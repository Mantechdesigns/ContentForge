# SEO Optimization Skill

## Purpose

Give OpenClaw search engine expertise — keyword research, meta optimization, content structuring for rankings.

## When to Use This Skill

- User wants to optimize blog posts or landing pages for SEO
- User needs keyword research for a niche or topic
- User asks for meta titles and descriptions
- User wants content structured for featured snippets
- User needs internal linking strategy

## Capabilities

- Research keywords with intent classification (informational, transactional, navigational)
- Write SEO-optimized titles, H1s, meta descriptions
- Structure content with proper heading hierarchy for featured snippets
- Analyze competitor page structure and backlink profiles
- Suggest internal linking opportunities
- Generate schema markup for rich results

## Example Prompts

```
"Research the top 20 keywords for 'agency scaling' and classify by intent"
"Optimize this blog post for the keyword 'how to scale a digital agency'"
"Write meta titles and descriptions for my 5 landing pages"
"Analyze the top 3 ranking pages for 'business coaching' and identify gaps"
```
