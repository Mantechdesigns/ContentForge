# Funnel Hacking Skill

## Purpose

Teach OpenClaw to reverse-engineer any successful funnel and recreate it in your brand voice.

## When to Use This Skill

- User wants to analyze a competitor's funnel step-by-step
- User needs to replicate a winning funnel structure
- User asks for funnel audit or optimization
- User wants to build a new funnel based on proven templates

## Funnel Types

| Type | Best For | Pages |
|------|----------|-------|
| **Tripwire** | Low-ticket entry ($7-$47) | Ad → Landing → Checkout → Upsell |
| **VSL** | Mid-ticket ($97-$997) | Ad → VSL Page → Application → Call |
| **Challenge** | Community building | Ad → Registration → 5-Day Challenge → Offer |
| **Webinar** | High-ticket ($997-$5K) | Ad → Registration → Webinar → Application |
| **Book** | Authority building | Ad → Free + Shipping → Upsell Chain |

## Capabilities

- Map any competitor funnel from entry to final offer
- Identify conversion points and potential leaks
- Generate full funnel blueprints with page copy
- Calculate expected conversion rates per funnel stage
- Suggest traffic sources and ad strategies per funnel type

## Example Prompts

```
"Reverse-engineer Alex Hormozi's $100M Leads funnel step by step"
"Build me a webinar funnel for my $2,000 coaching program"
"Analyze this landing page URL and identify the funnel structure"
"What funnel type should I use for a $497 digital course?"
```
