# Copywriting Mastery Skill

## Purpose

Make OpenClaw a world-class copywriter — sales pages, emails, ads, and persuasive content using proven frameworks.

## When to Use This Skill

- User needs a sales page or VSL script
- User wants email sequences (welcome, nurture, launch, abandoned cart)
- User asks for ad copy (Facebook, Google, TikTok, LinkedIn)
- User wants landing page copy optimized for conversions
- User needs persuasive content for any medium

## Frameworks

| Framework | Best For |
|-----------|----------|
| **AIDA** | Ads, short-form copy |
| **PAS** | Pain-driven offers |
| **BAB** | Before-After-Bridge transformation stories |
| **4 P's** | Promise, Picture, Proof, Push for sales pages |
| **Star-Chain-Hook** | Storytelling-based persuasion |
| **HSO** | Hook-Story-Offer for webinars and VSLs |

## Capabilities

- Write complete sales pages with proven structures
- Generate email sequences with subject line variants
- Create ad copy with multiple angles (pain, aspiration, proof, urgency)
- A/B test copy variations
- Analyze and improve existing copy for conversion rate
- Write in any brand voice when Brand Profile is loaded

## Example Prompts

```
"Write a sales page for a $997 coaching program using the 4 P's framework"
"Create a 7-day email nurture sequence for new leads"
"Write 10 Facebook ad variations — 5 pain-focused, 5 aspiration-focused"
"Rewrite this landing page to increase conversions, keeping my brand voice"
```
