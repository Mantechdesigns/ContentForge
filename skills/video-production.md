# Video Production Skill

## Purpose

Turn OpenClaw into an AI video producer — from script to final render, optimized per platform.

## When to Use This Skill

- User wants to produce a talking-head, B-roll, or cinematic video
- User needs voiceover scripts timed to specific durations (15s, 30s, 45s, 60s)
- User asks for B-roll recommendations for a topic
- User wants to batch-produce multiple videos at once
- User needs platform-specific formatting (9:16 for TikTok, 1:1 for Instagram)

## Video Formats

| Format | Duration | Platform | Best For |
|--------|----------|----------|----------|
| **Quick Hook** | 15s | TikTok, Reels | Attention-grabbing single point |
| **Story Reel** | 30s | TikTok, Reels | Problem → solution narrative |
| **Deep Dive** | 45-60s | YouTube Shorts | Educational content |
| **Long Form** | 3-10min | YouTube | Tutorials, case studies |
| **Carousel Video** | 30-60s | Instagram | Multi-point frameworks |

## Capabilities

- Script videos with word-count-to-duration mapping (150 words ≈ 60s)
- Auto-select B-roll from asset library or generate via AI
- Format for any platform aspect ratio
- Generate voiceover via ElevenLabs or HeyGen
- Batch-produce 1, 3, 5, or 7 videos at once
- Export to AutoPilot queue for scheduling

## Example Prompts

```
"Produce a 30-second TikTok about profit leaks using my AI clone"
"Batch-produce 5 talking-head videos about the BRR framework"
"Script a 60-second YouTube Short with B-roll overlay"
"Convert this long-form video into 3 short-form clips"
```
