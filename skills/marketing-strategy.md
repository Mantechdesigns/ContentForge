# Marketing Strategy Skill

## Purpose

Equip OpenClaw with expert-level marketing strategy capabilities — from funnel architecture to offer creation to ad optimization.

## When to Use This Skill

- User wants to design a marketing funnel (VSL, challenge, webinar, tripwire)
- User needs help positioning an offer or creating an irresistible offer stack
- User asks for ad copy, landing page copy, or email sequence
- User wants to analyze competitor funnels
- User needs campaign planning for a product launch

## Frameworks Included

| Framework | Purpose |
|-----------|---------|
| **Value Ladder** | Structure offers from free → high-ticket |
| **Hook-Story-Offer** | Russell Brunson's 3-part content formula |
| **Dream 100** | Identify top distribution channels |
| **AIDA** | Attention → Interest → Desire → Action copy framework |
| **PAS** | Problem → Agitation → Solution for pain-driven copy |

## Capabilities

- Generate complete funnel blueprints with page-by-page copy
- Write offer stacks with pricing psychology
- Create ad variations (Facebook, Google, TikTok formats)
- Audit existing funnels for conversion leaks
- Plan email nurture sequences (5-day, 7-day, 14-day)

## Example Prompts

```
"Design a webinar funnel for a $2,000 coaching program targeting agency owners"
"Write 5 Facebook ad variations using the PAS framework for my course"
"Audit my landing page and identify the top 3 conversion leaks"
"Create an irresistible offer stack for my $497 program"
```
