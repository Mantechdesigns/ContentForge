# Viral Content Creation Skill

## Purpose

Make OpenClaw a viral content machine — hooks, captions, carousel scripts, and engagement tactics backed by data.

## When to Use This Skill

- User wants viral hooks for Reels, TikToks, or Shorts
- User needs captions optimized for engagement
- User wants carousel scripts (5-slide, 7-slide, 10-slide)
- User asks for content repurposing across platforms
- User wants to analyze what makes content go viral

## Hook Formulas

| Formula | Example |
|---------|---------|
| **Provocative Statement** | "If you're not making $10K/month, you have a skills problem" |
| **Curiosity Gap** | "I made $500K doing THIS one thing (and it's not what you think)" |
| **Contrarian Take** | "Stop posting daily on social media — here's why" |
| **Direct Challenge** | "You're losing money every day you don't do this" |
| **Shocking Statistic** | "97% of coaches fail in their first year. Here's how to be the 3%" |
| **Story Hook** | "Last year I was broke. Today I run a 7-figure agency. Here's what changed" |

## Capabilities

- Generate 10-30 hook variations for any topic
- Write platform-specific captions (TikTok vs Instagram vs LinkedIn)
- Create carousel frameworks with slide-by-slide copy
- Analyze Viral Intelligence data and replicate winning patterns
- A/B test hook variations for maximum engagement

## Example Prompts

```
"Give me 20 hook variations for a video about scaling past $100K/month"
"Write a 7-slide Instagram carousel about the BRR framework"
"Repurpose this YouTube video into 5 TikTok scripts"
"Analyze the top 5 viral posts and create similar content in my brand voice"
```
